deriveScenegraph(scenegraphNodeName: string, gltf: any)

This function derives a scenegraph from a GLTF file based on the provided scenegraph node name. It traverses the scenes in the GLTF file to find the specified node and updates its transformation matrix.

Parameters :
Name Type Optional Description
scenegraphNodeName string No

The name of the scenegraph node to be derived.
gltf any No

The GLTF file data.
Returns : any

The updated GLTF file data with the derived scenegraph.
loadGLTF
loadGLTF(model: SpatialSceneNode, cache?)

This function loads a GLTF file from a given URL and parses it using the GLTFLoader. It also supports caching of the GLTF files. After loading, it derives the scenegraph for the specified node.

Parameters :
Name Type Optional Description
model SpatialSceneNode No

The spatial scene node containing the scenegraph URL.
cache Yes

An optional cache for storing GLTF file promises.
Returns : Promise<any>

A promise that resolves to the parsed GLTF file data with the derived scenegraph.
